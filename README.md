## 🛠 Tech Stack
HTML5 · CSS3 · Clerk (Google authentication) · AI-assisted interactivity (Gemini 3.5 Flash) · GitHub Pages

## 🎨 Design references
- [Refero.design](https://refero.design) (Minimalist grids, typography, and card actions)
- [Godly.website](https://godly.website) (Clean whitespace and elegant transitions)

## 🤖 How I used AI
I designed the minimalist layout, CSS typography structure, and UX routes; AI helped structure the initial product database, responsive queries, and Clerk mounting callback scripts.

## 📚 What I learned
During this project, I learned how to integrate Clerk Authentication into a purely vanilla HTML/JS application using the browser SDK, manage complex cart state transitions with localStorage, and implement mobile-first grid components without any CSS libraries.
